What is SmartMap?

SmartMap is a public-facing mapping platform that helps users:

  • Find nearby services such as schools and hospitals.
  • Explore locations using map-based interactions.
  • Access spatial data in a simple and accessible way.

How I Evaluated The Product

To fundamentally understand the underlying usability challenges, I took a structured approach:

  • Explored service discovery flows (e.g., finding local schools or hospitals).
  • Rigorously tested map interactions and active layer toggling logic.
  • Identified and mapped precise navigation friction points across the interface.
  • Evaluated baseline usability specifically for first-time general users.

UX Problems

  • Navigation overload
  • Layer complexity
  • Discoverability issues
  • High cognitive load

UX Decisions

1. Navigation Complexity

→ Introduced structured toolbar
→ Reduced confusion

2. Layer Overload

→ Grouped layers with strict hierarchy
→ Improved clarity

3. Service Discovery

→ Added category-based filtering
→ Faster task completion

4. Cognitive Load

→ Applied progressive disclosure
→ Cleaner map experience

5. First-Time Use

→ Added onboarding cues & tooltips
→ Easier learning curve
